History

What is History

History save all of the commands you ran (even if they failed or do not exist) to a file called .history.krabb and it would print it line by line and add numbers to every command.

How does it work

The History command works like this:

When you are running a function (before executed) it is written to the .history.krabb file, and when you run the History command it prints the file line by line until it gets to the end of the file.

Why should I use it

The only official way to view your command history in krabb is with the history command, so if you want to view your command history - the history command is the only official way.

How to clear the Command History

To clear the command history you can just open your home directory and open the .history.krabb (hidden file) file and just remove everything in there.

You can also remove it with this command (you dont need krabb shell to execute it):

➜ downloads cd ~
➜ ~ echo "" > .history.krabb

The History file

The history file is located at the home directory of your computer and it is called .history.krabb it can store infinite commands in it.

How To use it

In the Shell

To run the History command in the shell you should do this:

history

Your output should look like this:

{} -> history
 1 history
{} ->

In a Script

It is not reccommended to use the History command in a Script because it can sometimes print a long list of commands. In the current version you cannot fetch data from the history command.
